구글 마이맵에 넣는 법
  1. PC 브라우저에서 google.com/maps/d → 새 지도 만들기
  2. 왼쪽 레이어 박스의 [가져오기] → 이 CSV 업로드
  3. 위치 열 = 주소, 마커 제목 열 = 관리번호(또는 이름) 선택
  4. 폰에서 보기: 구글맵 앱 → 저장됨 → 지도
데이터는 서버로 전송되지 않습니다. 새로고침하면 모두 사라집니다.
